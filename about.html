    저는 빠르게 변화하는 웹 기술과 사용자 경험에 대한 최신 트렌드를
    한발 앞서서 받아들일 수 있는 웹 개발자가 되고자 합니다.
    JS 프레임워크, 라이브러리, 그리고 CSS4와 같은 새로운 웹 개발
    툴들이 지속적으로 등장하고 발전하고 있습니다. 이러한 동향을
    주시하며 최신 기술을 습득하고, 사용자들에게 더 나은 경험을 제공
    할 수 있는 웹 개발자로 성장하고자 합니다.

    1. Vanilla JS

      Udemy에서 바닐라 JavaScript 강의를 구매하여 수강하고 있습니다. 이 강의를 통해 React, Vue.js와 같은 JavaScript 프레임워크 및 라이브러리의 기초를 이해하고자 합니다. 또한, GSAP의 다양한 애니메이션을 JavaScript로 구현하는 것이 제 목표입니다.

    2. React

      웹 개발 분야에서 가장 인기 있는 라이브러리 중 하나인 React를 학습하고 있습니다. React를 배워 현재의 웹 개발 스킬을 더욱 향상시키고, 더 다양한 프로젝트에 참여할 수 있는 기회를 갖고 싶습니다.

    3. TypeScript

      React 프로젝트를 더 안정적으로 만들기 위해 TypeScript를 함께 학습하려고 합니다. TypeScript를 사용함으로써 코드의 안정성과 가독성을 향상시키고, 자동 완성 기능을 활용하여 개발 효율성을 높일 수 있습니다. 또한, 코드 품질을 향상시키는 장점들을 활용하여 높은 퀄리티의 프로젝트를 구축하고자 합니다.
